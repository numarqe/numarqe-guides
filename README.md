# Numarqe Guides

Open finance operations guides, AP playbooks, buyer's guides and vendor comparisons by [Numarqe](https://numarqe.com/).

This repository exists for two reasons:
- publish genuinely useful finance operations content in a clean, indexable markdown format
- create durable references back to [numarqe.com](https://numarqe.com/) and core product pages

It is intentionally not a brochure mirror. The goal is to publish practical, comparison-led and operations-led content that helps finance teams evaluate tools, tighten controls and modernise payables.

## Publishing model

This repo is set up so markdown pages in `docs/` can publish to GitHub Pages.

Recommended workflow:
1. Open an issue for a page, brief or review task.
2. Create a branch from that issue.
3. Draft or edit markdown.
4. Open a pull request.
5. Merge to `main`.
6. GitHub Pages deploys the merged markdown automatically.

On GitHub these are pull requests, not merge requests. If your team says “MR” internally, treat that as “PR” in this repo.

## Editorial standards

- useful before promotional
- comparison-led, checklist-led and buyer-intent content
- clear links back to https://numarqe.com/
- avoid copying the marketing site word-for-word
- add original tables, checklists, frameworks and decision criteria
- keep tone practical and credible rather than hype-heavy
